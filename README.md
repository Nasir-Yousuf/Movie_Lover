# Movie Watchlist App

A React movie tracking application that allows users to search movies, select them, and maintain a watched list with persistent state using localStorage.

## Features

- Search movies using an external API
- Select and deselect movies
- Add and remove movies from a watched list
- Persistent watched list using browser localStorage
- Custom reusable `useLocalStorageState` hook

## Tech Stack

- React
- JavaScript (ES6+)
- Browser localStorage

## What I Learned

- Building reusable custom React hooks
- Managing state with `useState`
- Synchronizing state with side effects using `useEffect`
- Persisting application data using localStorage
- Writing cleaner and more predictable state update logic
